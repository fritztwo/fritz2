Before moving on to more specialized and occasionally niche topics, we want to pause for a moment to **recap** what 
we have learned so far.

The core foundations of every fritz2 application include:
* **Reactive rendering** of data
* **Data management**, including mapping data to the appropriate form
* The use and **reactive binding of events** to the data management layer
* **Validation** of the entered data

These topics have been covered in detail and at various levels of depth. With these skills, you are already equipped 
to build non-trivial applications.

Information is still pending regarding:
* **Routing** within a single page application (SPA)
* Integrating **HTTP API calls**
* Optional topics like **WebSockets** or even **WebComponents**

Depending on your requirements, these topics are important, but they no longer fit neatly into the established 
framework used for the foundations (Overview, Basics, and Advanced topics).

Therefore, the following chapters should always be viewed as self-contained entities.

With the knowledge you’ve gained, you can now already dive into using the [headless components](/headless).
